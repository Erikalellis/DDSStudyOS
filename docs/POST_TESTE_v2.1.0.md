Pessoal, liberamos uma nova versao de testes do **DDS StudyOS v2.1.0**.

Melhorias desta versao:
- Correcao de abertura do app apos instalacao.

- Novo icone oficial do DDS StudyOS.
- Instalador com verificacao de pre-requisito e download automatico do `.NET Desktop Runtime 8 (x64)` quando necessario.
