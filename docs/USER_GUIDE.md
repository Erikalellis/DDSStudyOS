# DDS StudyOS - Guia Rapido de Uso

Este guia resume os fluxos principais para uso diario do aplicativo.

## 1) Primeiro acesso

1. Abra o app e conclua o onboarding inicial.
2. Confira seu perfil em `Configuracoes`.
3. Defina sua rotina de estudo (meta diaria e preferencias de pomodoro).

## 2) Cursos

1. Acesse `Cursos`.
2. Cadastre nome, plataforma e URL.
3. Marque favoritos para acesso rapido.
4. Use export/import de favoritos por perfil quando necessario.

## 3) Materiais e certificados

1. Acesse `Materiais & Certificados`.
2. Salve caminho local, URL ou copia gerenciada.
3. Use filtros por:
- curso;
- tipo;
- data (inicio/fim).
4. Use `Abrir pasta` para localizar o material.
  O app aplica fallback automatico quando o arquivo nao existir mais.

## 4) Agenda

1. Acesse `Agenda`.
2. Cadastre lembretes por curso.
3. Monitore os proximos lembretes no `Home`/`Dashboard`.

## 5) Navegador interno

1. Acesse `Navegador`.
2. Use `dds://inicio` para pagina inicial interna.
3. Use favoritos e atalhos para abrir cursos e links salvos.

## 6) Desenvolvimento e atualizacao

1. Acesse `Desenvolvimento`.
2. Escolha canal `stable` ou `beta`.
3. Use `Verificar agora` e `Atualizar agora` para aplicar release online.
4. O app valida integridade (`SHA256`) e assinatura configurada.

## 7) Backup e diagnostico

1. Acesse `Configuracoes`.
2. Gere backup e valide antes de restaurar.
3. Exporte o bundle de diagnostico (`.zip`) para suporte.
